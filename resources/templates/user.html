Generate and Download your Authkey

This will generate a new authkey that you can use to login to SDoS servers. You need this in order to be able to play in the tournament. Note: this is not the same as an "Auth" authkey that is used to get master on servers. It is only used to log in to SDoS, and it's tied to your SDoS account.

When you generate the key, it will download a file that contains your authkey called "once.cfg". Please copy this file into your Sauerbraten directory, and your authkey will be automatically loaded.

Where to put the file:

  • Windows: My Documents>My Games>sauerbraten
  • Linux: ~/.sauerbraten (but you already knew that didn't you, you linux user you)
  • Mac: home/Library/Application Support/sauerbraten

You should now automatically log on when you join an SDoS server.
